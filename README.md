# Лабораторна робота №3 

## Тема
Git та Linux CLI

## Модель
Математичне моделювання біологічного росту бактерій у замкненому середовищі 

## Автор
Бордіян Микола Павлович, група АІ-231

## Опис
Реалізовано систему моделювання динаміки популяції бактерій у замкненому просторі. 
В основі лежить **система звичайних диференціальних рівнянь (ЗДР)**, що використовує **рівняння Моно** для опису лімітування росту поживним субстратом.

**Ключові показники:**
* Розрахунок питомої швидкості росту ($\mu$).
* Моделювання споживання субстрату ($S$) через коефіцієнт виходу ($Y$).
* Візуалізація переходу від експоненціальної до стаціонарної фази росту.

## Параметри системи
| Параметр | Значення | Опис |
| :--- | :--- | :--- |
| $\mu_{max}$ | 0.5 | Макс. швидкість росту (год⁻¹)|
| $K_s$ | 0.01 | Константа напівнасичення (г/л)|
| $Y$ | 0.4 | Ефективність використання субстрату|
